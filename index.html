Pascal versions of Add 3 numbers

To make things simple often requires complexity


This is a Pascal project to demonstrate the NSW Software Design & Development syllabus 8.2.1 Planning & Designing efficient algorithms.

Changes

1.0.1

The initial version does not add the numbers, but concatenates them

1.0.2 

Entries are converted to numbers and now ADD

1.0.3

The client wanted a product of the 3 numbers and concatenated sentence for the sum

1.0.4

Entries are now validated as integers after the final entry (on submit)

1.0.5

The client now only wants even positive numbers less than 40. Repeated validation code is added. The repetition of validation amounts to 95 lines

1.0.6 

A validation function is created so the same code can be called 3 times - back to 45 lines.

1.0.7 

Instead of just adding 3 numbers, we can keep adding numbers by typing "y"
  To make this simpler, a user could enter numbers, finishing when they add 0
